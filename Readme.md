## 前言
- 为什么有这个仓库？本人在学习强化学习理论和扩散模型遇到很多数学问题，我现在觉得很多基础数学知识需要弥补，所以查了以下应用数学和数理统计的培养方案，删除了几何，组合，数论，图论，拓扑学这些我觉得不太相关的内容，根据学习路线按顺序排列。
- 文件夹内包含我学习的笔记，学习课程是网站，视频，教科书等资料汇总，我尽量后面全用latex书写笔记。
- 参考网站 [CS自学指南](https://csdiy.wiki/), [同济大学数学培养方案](https://math.tongji.edu.cn/bkspy/zyjs.htm), [北京大学数院培养方案](https://www.math.pku.edu.cn/bks/pyfa/index.htm) , [统计学自学指南](https://xuankaiwang.github.io/)
